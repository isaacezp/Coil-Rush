# COIL RUSH

> A polished, high-speed snake remix.

Engineered for performance, featuring a continuous 60fps canvas loop and modern game feel.

## TECHNICAL ARCHITECTURE

* **Render Engine:** Custom 60fps canvas loop with native support for keyboard, pointer, and multitouch events.
* **State Management:** Full lifecycle handling (start, pause, crash, game-over, instant-restart).
* **Persistence:** Local top-5 leaderboard integration via web storage.

## GAMEPLAY & MECHANICS

* **Dynamics:** High-risk boost/combo multipliers, escalating speed curves, mine hazards, and precise self-collision detection.
* **Game Feel (Juice):** Integrated particle systems, screen shake, hit-stop frames, ripples, visual flashes, and reactive synth audio.

## ADAPTABILITY & A11Y

* **Cross-Platform:** Fluid scaling across desktop, as well as portrait and landscape mobile viewports.
* **Accessibility:** Built-in reduced-motion support, automatic pause-on-hidden, and highly accessible control schemes.
